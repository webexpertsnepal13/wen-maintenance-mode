The fastest & simplest maintenance page for WordPress site. Super-easy to use!

== Description ==

Create an Maintenance Mode Page that takes **less than a minute to install & configure**. This plugin won't waste your time! It is simple, fast, efficient & user-friendly with just four tab settings.

**Tabs and Options**
'General' Tab
  * Enable Maintenance Mode
  * Choose Template
  * Display Logo
  * Logo
  * Background Options
'Content' Tab
  * Heading
  * Content
  * Content Border
  * Choose Border Color
  * Content Text Color
'Social' Tab
  * Facebook
  * Twitter
  * LinkedIn
  * Instagram
  * YouTube
  * Email
  * Phone Number
  * Social Icon color
'Misc' Tab
  * Page Title
  * Favicon
  * Google Analytics Tracking
  * Google Tracking ID


== Installation ==

Follow the usual routine;

1. Open WordPress admin, go to Plugins, click Add New
2. Enter "WEN Maintenance Mode" in search and hit Enter
3. Plugin will show up as the first on the list, click "Install Now"
4. Activate & open plugin's settings page located under the Settings menu

Or if needed, upload manually;

1. Download the plugin.
2. Unzip it and upload to "/wp-content/plugins/"
3. Open WordPress admin - Plugins and click "Activate" next to the plugin
4. Activate & open plugin's settings page located under the Settings menu
